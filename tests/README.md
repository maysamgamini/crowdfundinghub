# tests

## Purpose
Contains the automated test projects that protect the architecture, unit behavior, and future integration scenarios.

## Files
- No direct files live in this folder; it primarily organizes child folders.

## Child Folders
- `ArchitectureTests`: Groups tests that enforce clean-architecture boundaries and project reference rules.
- `IntegrationTests`: Groups integration tests for end-to-end or component-level scenarios.
- `UnitTests`: Groups fast-running unit tests that exercise domain and application behavior in isolation.
