# Events

## Purpose
Contains shared abstractions for publishing and handling application events.

## Files
- `BaseApplicationEvent.cs`: Application event contract published for cross-module reactions.
- `IEventHandler.cs`: Application event handler for the surrounding use case.
- `IEventPublisher.cs`: Supporting source file for I Event Publisher.
- `NotificationRegistrationExtensions.cs`: Extension methods that encapsulate repeated registration or configuration logic.

## Child Folders
- None.
