# Crowd Funding Building Blocks Domain

## Purpose
Provides reusable domain primitives such as base entities, domain events, and value objects.

## Files
- `CrowdFunding.BuildingBlocks.Domain.csproj`: Project file that defines dependencies, target framework, and assembly references for this area.

## Child Folders
- `Common`: Contains base domain building blocks used across aggregates.
- `ValueObjects`: Contains reusable immutable value objects shared by multiple modules.
