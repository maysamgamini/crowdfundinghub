# Contributions

## Purpose
Defines API contracts for contribution endpoints.

## Files
- `ConfirmContributionPaymentRequest.cs`: Request contract consumed by an API endpoint.
- `ConfirmContributionPaymentResponse.cs`: Response contract returned by an API endpoint.
- `FailContributionPaymentRequest.cs`: Request contract consumed by an API endpoint.
- `FailContributionPaymentResponse.cs`: Response contract returned by an API endpoint.
- `ListContributionsResponse.cs`: Response contract returned by an API endpoint.
- `MakeContributionRequest.cs`: Request contract consumed by an API endpoint.
- `MakeContributionResponse.cs`: Response contract returned by an API endpoint.
