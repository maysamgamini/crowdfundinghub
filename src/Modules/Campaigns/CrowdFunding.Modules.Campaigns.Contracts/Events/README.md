# Events

## Purpose
Groups event contracts or domain events used by the Campaigns module.

## Files
- No direct files live in this folder; it primarily organizes child folders.

## Child Folders
- `CampaignCancelled`: Contains source files related to Campaign Cancelled.
- `CampaignCreated`: Contains source files related to Campaign Created.
- `CampaignPublished`: Contains source files related to Campaign Published.
