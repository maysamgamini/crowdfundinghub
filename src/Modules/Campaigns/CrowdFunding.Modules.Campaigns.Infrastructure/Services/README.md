# Services

## Purpose
Contains infrastructure service implementations used by the Campaigns module.

## Files
- `CampaignContributionAvailabilityReader.cs`: Query-side adapter used to retrieve data across boundaries.
- `CampaignReadService.cs`: Read-model service used by query handlers to fetch denormalized data.
- `SystemDateTimeProvider.cs`: Abstraction or implementation for retrieving the current UTC time.

- None.
