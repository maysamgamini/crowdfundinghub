# Crowd Funding Modules Campaigns Domain

## Purpose
Contains the core domain model for the Campaigns module, including invariants, events, and enums.

## Files
- `CrowdFunding.Modules.Campaigns.Domain.csproj`: Project file that defines dependencies, target framework, and assembly references for this area.

## Child Folders
- `Aggregates`: Contains the aggregate roots that protect the core business rules for the Campaigns module.
- `Enums`: Contains enum values that describe state transitions and classifications in the Campaigns module.
- `Events`: Groups event contracts or domain events used by the Campaigns module.
