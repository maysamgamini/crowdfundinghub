# Campaigns

## Purpose
Contains source files related to Campaigns.

## Files
- No direct files live in this folder; it primarily organizes child folders.

## Child Folders
- `Commands`: Groups contract-level commands exposed by the Campaigns module.
- `Events`: Contains application event handlers that react to cross-module events relevant to the Campaigns module.
- `Queries`: Groups contract-level queries exposed by the Campaigns module.
