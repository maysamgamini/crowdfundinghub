# Queries

## Purpose
Groups contract-level queries exposed by the Campaigns module.

## Files
- No direct files live in this folder; it primarily organizes child folders.

## Child Folders
- `GetCampaignById`: Implements the Get Campaign By Id query workflow for the Campaigns module.
- `GetCampaignContributionAvailability`: Implements the Get Campaign Contribution Availability query workflow for the Campaigns module.
- `ListCampaigns`: Implements the List Campaigns query workflow for the Campaigns module.
