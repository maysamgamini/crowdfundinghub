# Campaign Reviews

## Purpose
Contains source files related to Campaign Reviews.

## Files
- No direct files live in this folder; it primarily organizes child folders.

## Child Folders
- `Commands`: Groups contract-level commands exposed by the Moderation module.
- `Events`: Contains application event handlers that react to cross-module events relevant to the Moderation module.
- `Queries`: Groups contract-level queries exposed by the Moderation module.
