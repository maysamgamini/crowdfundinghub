# Crowd Funding Modules Moderation Domain

## Purpose
Contains the core domain model for the Moderation module, including invariants, events, and enums.

## Files
- `CrowdFunding.Modules.Moderation.Domain.csproj`: Project file that defines dependencies, target framework, and assembly references for this area.

## Child Folders
- `Aggregates`: Contains the aggregate roots that protect the core business rules for the Moderation module.
- `Enums`: Contains enum values that describe state transitions and classifications in the Moderation module.
- `Events`: Groups event contracts or domain events used by the Moderation module.
