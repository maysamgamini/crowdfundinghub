# Persistence

## Purpose
Groups persistence concerns for the Moderation module, including EF Core configuration, db contexts, repositories, and migrations.

## Files
- No direct files live in this folder; it primarily organizes child folders.

## Child Folders
- `Configurations`: Contains EF Core entity configuration classes for the Moderation module.
- `DbContexts`: Contains EF Core database contexts for the Moderation module.
- `Migrations`: Contains the EF Core migration history for the Moderation module. These files are generated snapshots of schema evolution.
- `Repositories`: Contains repository implementations for the Moderation module.
