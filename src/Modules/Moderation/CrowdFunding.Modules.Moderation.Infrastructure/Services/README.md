# Services

## Purpose
Contains infrastructure service implementations used by the Moderation module.

## Files
- `CampaignReviewReadService.cs`: Read-model service used by query handlers to fetch denormalized data.
- `CampaignReviewStatusReader.cs`: Query-side adapter used to retrieve data across boundaries.
- `SystemDateTimeProvider.cs`: Abstraction or implementation for retrieving the current UTC time.
