# Crowd Funding Modules Moderation Contracts

## Purpose
Contains integration-facing contracts for the Moderation module, such as cross-module commands, queries, and events.

## Files
- `CrowdFunding.Modules.Moderation.Contracts.csproj`: Project file that defines dependencies, target framework, and assembly references for this area.

## Child Folders
- `Commands`: Groups contract-level commands exposed by the Moderation module.
- `Events`: Groups event contracts or domain events used by the Moderation module.
- `Queries`: Groups contract-level queries exposed by the Moderation module.
