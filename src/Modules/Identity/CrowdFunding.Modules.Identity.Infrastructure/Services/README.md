# Services

## Purpose
Contains infrastructure service implementations used by the Identity module.

## Files
- `JwtAccessTokenProvider.cs`: Creates signed JWT access tokens for authenticated users.
- `JwtOptions.cs`: Configuration model for JWT token generation.
- `Pbkdf2PasswordHasher.cs`: Hashes and verifies passwords using PBKDF2.
- `SystemDateTimeProvider.cs`: Abstraction or implementation for retrieving the current UTC time.

## Child Folders
- None.
