# Authorization

## Purpose
Contains shared authorization constants and claim metadata for the identity model.

## Files
- `CustomClaimTypes.cs`: Shared claim type names used in authentication and authorization.
- `PermissionConstants.cs`: Static constants shared across the solution.
- `RoleConstants.cs`: Static constants shared across the solution.
- `RolePermissionCatalog.cs`: Static catalog that maps higher-level concepts to their configured constants.

## Child Folders
- None.
