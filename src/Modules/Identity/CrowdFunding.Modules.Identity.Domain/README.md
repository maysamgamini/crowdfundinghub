# Crowd Funding Modules Identity Domain

## Purpose
Contains the core domain model for the Identity module, including invariants, events, and enums.

## Files
- `CrowdFunding.Modules.Identity.Domain.csproj`: Project file that defines dependencies, target framework, and assembly references for this area.

## Child Folders
- `Aggregates`: Contains the aggregate roots that protect the core business rules for the Identity module.
- `Entities`: Contains supporting entities that belong to the Identity domain model.
