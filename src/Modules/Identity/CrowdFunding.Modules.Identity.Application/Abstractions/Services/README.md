# Services

## Purpose
Defines service contracts that the Identity application layer relies on for time, reads, or externalized behavior.

## Files
- `AccessToken.cs`: Supporting source file for Access Token.
- `IAccessTokenProvider.cs`: Supporting source file for I Access Token Provider.
- `IIdentityDateTimeProvider.cs`: Abstraction or implementation for retrieving the current UTC time.
- `IPasswordHasher.cs`: Supporting source file for I Password Hasher.

## Child Folders
- None.
