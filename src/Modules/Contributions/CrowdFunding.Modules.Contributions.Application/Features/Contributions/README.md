# Contributions

## Purpose
Contains source files related to Contributions.

## Files
- No direct files live in this folder; it primarily organizes child folders.

## Child Folders
- `Commands`: Groups contract-level commands exposed by the Contributions module.
- `Events`: Contains application event handlers that react to cross-module events relevant to the Contributions module.
- `Queries`: Groups contract-level queries exposed by the Contributions module.
